# italoBot
Italo telegram bot

###Usage

1. 'npm install'
2. 'node index.js <botToken>'

tmp folder MUST exist

###Commands
* tette: download and send a picture from /r/tits
* figa: download and send a picture from /r/realGirls
* culo: download and send a picture from /r/ass
